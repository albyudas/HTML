HTML

¿QUÉ ES?

HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet.Las siglas corresponden a HyperText Markup Language,que podría ser traducido como Lenguaje de Formato de Documentos para Hipertexto.

USOS

Su principal uso es estructurar y presentar el contenido para la web, es decir, determina el contenido de la página web, pero no su funcionalidad. esta función la realizamos mediante los "markup" o marcado para anotar textos, imágenes, y otros contenidos que se muestran en el Navegador Web. El lenguaje de marcado HTML incluye "elementos" especiales tales como head, title, body, header, article, section, p, div, span, img , y otros muchos.

Otras tecnologías distintas de HTML son usadas generalmente para describir la apariencia y presentación de una página web (CSS) o su funcionalidad (JavaScript). HTML le da "valor añadido" a un texto estándar en español. Hiper Texto hace referencia a enlaces que conectan una página web con otro lugar dentro de la misma página u otra diferente. Esto hace que los vínculos sean un elemento fundamental de la Web. Al subir contenido a Internet y vincularlo a páginas de otras personas, te haces participante activo de esta Red Mundial.

NOVEDADES

Tras la creación del HTML5 podemos llegar más lejos usando menos recursos. Además, algunos lenguajes, como XHTML, resultan innecearios y quedan inutilizados. Los efectos del HTML5 ya nos están afectando en nuestro día a día. Un claro ejemplo de esto sería Google Chrome, un uscador muy utilizado y creado con HTML5. Aún así, todavía no podemos olvidar HTML4 ya que es a través de Internet Explore (HTML4) dónde descargamos Google Chrome.

Las principales novedades están destinadas a la forma de construir una web. Entre ellas destacamos la capacidad de insertar archivos multimedia, que tendrán etiquetas HTML para poder ser incluidos. Esto nos da la posibilidad de no tener que depender de plug ins para poder ver una determinada web. En resumen, HTML5 no se diferencia mucho de su predecesor HTML4 pero la principal novedad es el nivel de sofisticación del codigo que construimos a partir de HTML5.

CSS

INTRODUCCIÓN

“Hojas de estilo en cascada”. Es un lenguaje que permite aplicar una estructura a una página, es decir se usa para el diseño visual de la propia página. En CSS se puede definir colores, márgenes, tipos de letra, etc. También puede lograr que muchos documentos HTML compartan la apariencia, utilizando una única hoja de estilo para todos (que se especifica en un archivo .css). Gracias a esta particularidad, se evita tener que repetir el código en la estructura.

PARA EMPEZAR

Las Hojas de Estilo en Cascada se escriben dentro del código HTML de la página web, solo en casos avanzados se pueden escribir en un archivo a parte y enlazar la página con ese archivo. Para empezar, la manera más sencilla de aplicar estilos en con el atributo style en HTML.

Separar la definición de los contenidos y la definición de su aspecto presenta numerosas ventajas, ya que obliga a crear documentos HTML/XHTML bien definidos y con significado completo (también llamados "documentos semánticos"). Además, mejora la accesibilidad del documento, reduce la complejidad de su mantenimiento y permite visualizar el mismo documento en infinidad de dispositivos diferentes.

AGRUPACIONES

Para agrupar un texto, imágenes, tablas, se puede utilizar la etiqueta < div> .
Esta etiqueta agrupa a un determinado contenido y todo lo encerrado se convierte en un bloque. Al final del texto salta a la siguiente línea ya que es un bloque.

Seleccionar por ID identificador sirve para concretar entre los demás. Cualquier imagen,párrafo,etc, si se aplica un estilo, solo a los elementos que tengan esa ID tendrán el estilo concreto para esa ID.
Ejemplo: < p id=”ejemplo”> este ID debe ser descriptivo para identificar rápidamente. No puede haber espacios entre las comillas. En HTML se llama con #ejemplo{[...]} y dentro de los corchetes definiremos el estilo que queremos el párrafo.

Si hay párrafos, títulos, etc que tengan algo en común, podemos agruparlos para que tengan las mismas características. Utilizamos la etiqueta < class> .
Por ejemplo en un párrafo: < p class="etiqueta"> En HTML sería .etiqueta{[...]} que contiene el estilo para más de un elemento de nuestra página.

La etiqueta span : sirve para agrupar selecciones de texto.
Why I < span class="lovey-dovey">Love< /span> Cats .
En este ejemplo la palabra "Love" tendría dos estilos diferentes. Uno correspondiente a la clase lovey-dovey que definimos en el código, y el otro a la etiqueta de estilo span.

También se puede poner más de una clase para especificar más aún.
Ejemplo:
.texto
.ejemplo
< p class=”texto ejemplo”>
Aquí dentro del párrafo tendría la clase texto y ejemplo, pero los demás qué tengan clase texto no serán iguales a los del párrafo.
Es importante la separación entre ellos ya marca la suma de los estilos de una clase más otra.

Para cambiar dentro de otra etiqueta también podemos usar la propiedad descendiente:
p .ejemplo . Importante también el espacio para decir que es descendiente y solo cambiará la parte que queremos.

Podemos agrupar selectores añadiendo comas para ahorrar código: h1,h2,h3{[...]}

Las pseudoclases podemos verlas para los links que no están vistos, los que si están, etc.
Las pseudo-clases :link y :visited se pueden utilizar para aplicar diferentes estilos a los enlaces de una misma página:

  • :link se aplica a todos los enlaces que todavía no han sido visitados por el usuario.
  • :visited se aplica a todos los enlaces que han sido visitados al menos una vez por el usuario.

Las pseudo-clases :hover, :active y :focus permiten al diseñador web variar los estilos de un elemento en respuesta a las acciones del link que hace del usuario.

  • :hover se activa cuando el usuario pasa el ratón o cualquier otro elemento apuntador por encima de un elemento.
  • :active se acitva cuando pulsa con el ratón sobre un elemento.
  • :focus se activa cuando el elemento está seleccionado.

De mayor a menor especificidad están: ID,Clase Y Elemento.
Si son de misma importancia el navegador hace que el último sea el que se imponga sobre el otro.

Posición de elementos
  • position: relative: Posicinar el elemento con las propiedades: top, bottom, left, right.
  • position: absolute: Pone el elemento en cualquier parte de la pantalla.

Si queremos hacer como capas encima de otras utilizamos z-index . Este es el orden en el que debe dibujar los elementos.
z-index:1 es el más alejado y por ejemplo z-index:4 sería el que está más encima.

Un posicionamiento fijo,position: fixed deja el elemento en una posición en la pantalla aunque bajemos en la página web.

Podemos situar elemento al lado,abajo,etc de otro elemento con las etiquetas float:left(a la izquierda) y float:right (a la derecha).

Para poner espacios y que no quede ocupando espacio que no queremos se usa clear: both

JAVASCRIPT

INTRODUCCIÓN

JavaScript (JS) es un lenguaje de programación orientado a objetos para mejorar la interfaz entre el usuario y la página web. Es importante diferenciar entre JavaScript y Java, ya que, como diferencia fundamental JavaScript no necesita instalar ningún programa para incorporarlo a una página y que sea visualizado. Esta es una de las ventajas, pero también hay otra como la de que JavaScript es de un fácil manejo, ya que, el navegador es el que debe interpretar el lenguaje y no el usuario.

NACIMIENTO

¿Por qué nació JavaScript? Inicialmente, para realizar páginas, en el ámbito visual solo existía HTML y dedicado principalmente para textos. Se necesitaba crear páginas más interactivas con el usuario y por eso se creó JavaScript. En sus comienzos se creó para pequeños programas con Diseñadores Web y sin estar destinada a objetos, pero, la gran multitud de desarrollo le permitió al ser un lenguaje Opensource.

USOS

Realmente, la relación que tenemos con JavaScript es continua ya que su principal utilización es para introducir múltiples animaciones en los programas. En esta página, por ejemplo el efecto acordeón esta realizado con JavaScript.

Destacar, como ya hemos mencionado antes, la diferencia entre JavaScript del usuario y el del navegador. El usuario utiliza Navigator JavaScript y en cambio, el servidor para procesarlo, utiliza lo que recientemente se denomina LiveWire JavaScript.

En las páginas web lo podemos observar cuando insertamos relojes, fechas o para validar formularios sin tener que utilizar CGI y en Internet en Chat y Correos entre otros.
